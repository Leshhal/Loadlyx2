Phase 2.1 Toggle Patch

Included:
- Tenant-controlled toggles for promo banner, countdown timer, low stock indicator, free shipping messaging, and bundle discounts.
- Tenant Experience admin page now exposes ON/OFF checkboxes for each store conversion feature.
- Catalog, product page, and tenant landing now only show conversion widgets when enabled by the tenant.
- Added minimal checkbox-row styling.

Behavior:
- Conversion widgets are OFF unless enabled.
- Existing text values remain saved, but they no longer render unless the matching toggle is enabled.
