# Loadlyx Frontend - Phase 1

This is a Next.js app connected to the backend API.

Pages included:
- `/` platform landing page
- `/catalog` store and cart flow
- `/quote` move quote form + upsell results
- `/admin` simple admin for products, categories, quotes, and loads
- `/checkout/success`
- `/checkout/cancel`

Copy `.env.local.example` to `.env.local` before running.
